## 贡献指南

如果你想为 pnana 的开发做出贡献：

1. **查看 Issues**：了解当前需要解决的问题
2. **选择任务**：选择你感兴趣的功能或 bug
3. **提交 PR**：按照贡献指南提交 Pull Request
4. **参与讨论**：在 Issues 中参与功能讨论

### 如何参与

1. Fork 本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启 Pull Request
